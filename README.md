# AudioRecorder
This simple script for record audio

# Installing
Windows
```
1. Download exe file in releases
2. Run it!
```

Linux
```
$ git clone https://github.com/Sorami1/AudioRecorder
$ cd AudioRecorder
$ pip install -r requirements.txt
$ python AudioRecorder.py
```
